# 000NCL — Nextcloud + Dolibarr Agent Setup

Python scripts for provisioning multi-agent organisations on Nextcloud and Dolibarr — scoped shares, profiles, permissions, folder READMEs.

See `nextcloud-dolibarr-agent-setup/` for the full toolkit.

## What's inside
- Agent account creation (Nextcloud + Dolibarr + WordPress)
- /Shared/ folder structure with per-agent shares
- Nextcloud contact and profile population
- Dolibarr permission matrix setup
